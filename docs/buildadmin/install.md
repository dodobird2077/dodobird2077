# 快速安装

1. 克隆项目
```bash
git clone https://gitee.com/wonderful-code/buildadmin.git
```

2. 安装依赖
```bash
# 騰訊云鏡像
composer config -g repos.packagist composer https://mirrors.cloud.tencent.com/composer/
composer install
```

3. 启动项目 (启动后才配置数据库等信息)
```bash
php think run
```

4. 打开项目
```
http://127.0.0.1:8000/
```
